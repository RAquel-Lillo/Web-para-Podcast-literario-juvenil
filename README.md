# 🌐 Proyecto Web – Página HTML y CSS

## 📖 Descripción

Este proyecto consiste en el desarrollo de una **página web estática** creada con **HTML5** y **CSS3**. Su objetivo es aplicar los fundamentos del desarrollo web, la estructuración semántica del contenido y el diseño visual mediante hojas de estilo.

## ✨ Características

* Estructura HTML semántica.
* Diseño y estilos mediante CSS3.
* Organización clara y ordenada del contenido.
* Interfaz sencilla e intuitiva.
* Código limpio y fácil de mantener.
* Proyecto orientado al aprendizaje y la práctica del desarrollo web.
* Responsive

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3

## 🎯 Objetivo del proyecto

Este proyecto refleja mi aprendizaje en la creación de páginas web estáticas, aplicando buenas prácticas de estructuración del código y diseño de interfaces.

## 📚 Aprendizajes

Durante el desarrollo de este proyecto se han puesto en práctica conceptos como:

* Estructuración semántica de una página web.
* Uso de etiquetas HTML5.
* Aplicación de estilos con CSS3.
* Organización del contenido para mejorar la legibilidad.
* Separación entre estructura y presentación.
